# jiuyou-m-hub

## 项目简介

`jiuyou-m-hub` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这里不针对任何特定网站或域名，仅作为页面资源的集中存放与版本管理。

## 目录说明

```
.
├── index.html          # 仓库主页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   └── example2.html
├── assets/             # 页面所需的资源文件（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个文件为一个完整的 HTML 页面，可直接通过浏览器打开。
- `assets/`：存放页面引用的外部资源，保持文件结构清晰。

## 页面归档说明

- 本仓库仅用于归档静态 HTML 页面，不涉及后端逻辑或数据库。
- 每个页面应保持独立，尽量避免跨页面依赖。
- 如需引用公共资源，请统一放置在 `assets/` 目录下，并在页面中正确引用路径。

## 维护说明

- 欢迎通过 Pull Request 提交新的页面或改进现有页面。
- 提交前请确保页面功能正常，无外部不可访问资源。
- 请勿添加与页面内容无关的广告、追踪代码或恶意脚本。
- 本仓库不提供托管或部署服务，仅作为原始文件的版本控制与存档。
